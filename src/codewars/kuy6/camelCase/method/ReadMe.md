<h2>6 Kuy</h2>
<hr>
<p>Write simple <code>camelCase</code> method for strings. 
All words must have their first letter capitalized without spaces.</p>
<h3>Examples:</h3>
<pre>

camelCase("camel case word"); // => "CamelCaseWord"
